# DoraHacks Submission — Sentinel Treasury

## Project Name
Sentinel Treasury

## Tagline
The AI copilot for compliant on-chain treasury management

## Description
Sentinel Treasury is an AI-powered treasury management copilot built on HashKey Chain. It monitors on-chain treasury balances, analyzes market conditions via oracle feeds, and recommends optimal capital allocation between yield-generating vaults and liquid reserves — with full explainability for every decision.

Designed for DAOs, on-chain funds, and teams managing shared capital, Sentinel Treasury replaces manual treasury decisions with intelligent, policy-gated automation.

## Key Features
- **AI Recommendation Engine:** Analyzes oracle data, vault performance, and payout obligations to suggest optimal allocation
- **Explainable Decisions:** Every recommendation includes human-readable reasoning with scored metrics
- **Policy-Gated Approvals:** Authorized users confirm AI recommendations before on-chain execution
- **Payout Management:** Single-click treasury disbursements with reserve protection
- **Optional KYC Integration:** HashKey's native KYC SBT (isHuman) for compliance-aware gating
